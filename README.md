# tiny-routers
Simplify your REST router in few secs of setup
# Installation
`$ npm install --save tiny-routers`
# Usage
- Create your *.d.ts in root folder and add reference to `tiny-routers` type.
```javascript
// app.d.ts

/// <reference types="tiny-routers" />
```
